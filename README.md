# 🎓 MundiaBot - Assistant d'Orientation IA (RAG & LLM)

MundiaBot est un chatbot intelligent conçu pour l'Université Mundiapolis. Il automatise l'orientation académique en utilisant l'IA générative pour conseiller les étudiants selon leur profil.

## 🚀 Fonctionnalités clés
* **RAG Dynamique** : Extraction des filières en temps réel via Web Scraping sur le site officiel.
* **Intelligence Artificielle** : Utilisation de **LLaMA 3 (8B Instruct)** pour une analyse sémantique des profils.
* **Logique Métier** : Recommandation automatique de formations alternatives si les critères d'admission ne sont pas remplis.
* **Dashboard Admin** : Interface sécurisée pour visualiser les leads étudiants via SQLite.

## 🛠️ Installation et Lancement
1. Clonez le projet : `git clone https://github.com/votre-pseudo/Projet-MundiaBot.git`
2. Installez les dépendances : `pip install -r requirements.txt`
3. Configurez votre `HF_TOKEN` en variable d'environnement.
4. Lancez l'application
